Help

Welcome to the help page for Join, your guide to using our kanban project management tool. Here, we'll provide an overview of what Join is, how it can benefit you, and how to use it. Have fun with this tool.

What is Join?

Join is an agile Project Management Tool which is invented to bring Clarity to your work Process and to maximize the Efficieny of by limiting work in progress.

How to use it

1. The first step is to create a contact and enter all the required information. You can do this by clicking "Contacts" in the navigation. When you create a Contact there, it will appear in the "Add Task" Section. You can also edit and delete the Contacts.
2. The second Step is to create a Task and fill in all needed informations. You can do this by clicking "Add Task" in the navigation. The Contacts you created previously, will displayed in the drop down Menu "Assigned to". After you create a Task you will direct to the "Board".
3. The "Board" Section will display all created Task in a mini View. If you would like to get all the information about a task, click on the corresponding task. You can also Edit or delet the Tasks. If you click of one of the + Icons you can add a Task in the "Board" too. If you want to move the Tasks to a other Status, you can do it with drag and drop.
4. The "Summary" displays all information, such as the total number of tasks or how many urgent tasks there are and the next due date.

Enjoy using Join!
